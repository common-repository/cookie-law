== Description ==

The Cookie Law adds box message to the page on the first page view for each visitor. This plugin is used for implied consent and that means the visitor continues to the site and he agree to use cookies.

== Installation ==

1. Upload all files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the text through the 'Settings' menu under 'EU Cookie Message'
